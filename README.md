# 🔐 Password Manager

A simple **Password Manager application** built with **React** that allows users to store and manage their login credentials.
Users can create, edit, delete, and copy saved passwords easily from a clean and minimal interface.

This project focuses on practicing **React state management and UI interactions**.

## 📌 Project Overview

This Password Manager helps users manage their login credentials in one place.

Users can store information such as:

* Website URL
* Username
* Password

The interface allows users to quickly add new credentials, update existing ones, remove entries, or copy passwords when needed.

---

## ⚙️ Functionality

* Create password entries
* Edit existing entries
* Delete saved credentials
* Copy passwords quickly
* Clean and minimal UI

---

## 🎯 Purpose of the Project

This project was built to practice and demonstrate:

* React component structure
* State management using **useState**
* Side effects using **useEffect**
* Using **uuid** to generate unique IDs
* Notifications using **react-toastify**
